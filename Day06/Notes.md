## Notes

**Styles**
- style attribute is used to add style to elements
- style such as font size, color, and more
- syntax: \<tagname style = "property:value;" >
- Property and value are both part of the CSS file
---
- setting the background color of page: 
    - \<body style = "background-color:red;">
    - example: <body style = "background-color: grey;">
    - this actually sets the color of .md file to grey
- setting the background color of elements
    - \<h1 style = "background-color:red;">
    - example: <h1 style = "background-color:red" > hello </h1>
    - you can also do this with \<p> and other tags probably
- text color
    - style = "color:blue;"
    - example: <p style = "color: blue;"> hello </p>
    - again, you can do this with other tags 
- fonts
    - sytle = "font-family:verdana;"
- text size
    - style = "font-size:300%;"
- alignment
    - style = "text-align:center;"

**Formatting**
- you can add formatting tags to text
- example: <p><b>hello</b></p>
- \<b> to specify starting spot. \</b> to specify ending spot
- <strong>\<strong></strong> same as b
- <i>\<i></i>
- <em>\<em></em> same as i
- <mark>\<mark></mark> highlight
- <small>\<small></small> smol
- <del>\<del></del> strikethrough
- <ins>\<ins></ins> underline basically
- <sub>\<sub></sub> lower
- <sup>\<sup></sup> upper
